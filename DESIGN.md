This document defines the core aesthetic guidelines for our product, ensuring a consistent and harmonious user experience across all touchpoints.



## Theme Overview



The design system is currently set to a **light** color mode.



### Color Palette



Our brand identity is expressed through a carefully selected color palette:

- **Primary Color:** `#F8BBD0` - A light pink shade, used for primary actions and key interactive elements.

- **Secondary Color:** `#F8BBD0` - This matches the primary color, providing consistency across supporting elements.

- **Tertiary Color:** `#322705` - A dark, almost black, brown color serving as an accent for highlights or decorative elements.

- **Neutral Color:** `#FDF5E6` - A very light cream color, providing a soft background for surfaces and non-chromatic elements.



### Typography



We use a distinct typographic system to establish a clear visual hierarchy:

- **Headlines:** `notoSerif` - Provides a classic and elegant feel for prominent text.

- **Body Text:** `plusJakartaSans` - A modern, clean sans-serif for readability in longer passages.

- **Labels:** `plusJakartaSans` - Consistent with body text for clear and legible labels.



### Shape and Form



The roundedness of UI elements is set to **3**, resulting in maximum, pill-shaped corners for a soft and approachable aesthetic.



### Spacing



The design adopts a **compact** spacing (`2`), offering a balanced approach between visual breathing room and efficient use of space.
